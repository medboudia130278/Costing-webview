        About
        
      

      

        
Maintenance Costing Control Panel

        
Tool functionalities:

        
    
          
  1. Use two input files – one for non-linear assets and another for linear assets.
    2. 
          
  2. For linear assets, divide the line into several sections with their own kilometer points so that the file becomes structured similarly to the non-linear assets.
    3. 
          
  3. Plan maintenance activities for both linear and non-linear assets over an annual calendar, and calculate the required number of technicians (day and night) per subsystem.
    4. 
          
  4. Estimate the annual average cost of spare parts consumption throughout the maintenance contract.
    5. 
          
  5. Estimate the annual cost of subcontracting.
    6. 
          
  6. Estimate overhaul and renewal costs over the duration of the contract, considering asset lifetime and obsolescence shorter than the contract duration.
    7. 
          
  7. Generate an output file with a planning sheet for each type of activity (preventive, corrective, subcontracting, overhaul/renewal) and a summary sheet presenting the overall cost estimation.
    8.
